# ii-eve Translator

Sidebar translator page for the [ii-eve](https://github.com/djOB2EOTWQW1/ii-eve) Hyprland shell, packaged as an extension via the `sidebarLeftPages` contribution point. Previously a built-in policy page; now provided here.

Uses the [`translate-shell`](https://github.com/soimort/translate-shell) (`trans`) command-line tool. Translation target/source languages are still read from the shell's settings (`Config.options.language.translator.*`).

## Install

Extensions settings → reveal the path/URL input → paste this directory's absolute path (local) or the repo URL → Install → enable. The Translator tab appears in the left (policies) sidebar.
